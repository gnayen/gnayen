Accupass 選擇票券

Web ui / ux design

主要痛點

1. 選擇票券時，無法查看已購買票券張數及即將付款總金額，導致下一步付款時若反悔還得返回上一步

2. 優惠代碼邏輯已不符使用（ ex. 輸入優惠代碼後 1000元 -> 100元、5000元 -> 1000元 ）

對應解法

1. 選擇票券的同時告知數量及總金額

2. 優惠代碼邏輯更改 ( ex. 輸入優惠代碼後 1000元 -> 900元、5000元 -> 4900元 )，並移至付款流程中
