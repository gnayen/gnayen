Accupass 費用方案

Web ui / ux design & coding

主要痛點

1. 原費用方案只有三種選擇並且無具體購買後可達到的效益

2. 無管道可以聯繫購買

對應解法

1. 包成 A、B、C 方案各種搭配組合，並說明什麼類型用戶適合什麼服務並可以達成什麼效益。
另外的散客，則有散裝的服務以 Tag 推薦給不想購買套裝服務又毫無頭緒的散客

2. 添加聯繫管道並且浮動在網站角落，隨時可以聯繫購買
